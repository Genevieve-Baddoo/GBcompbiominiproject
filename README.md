
HCMV (Human cytomegalovirus) Mini Project.
This respisotory contains a python pipeline that takes SRA input reads from the HCMV virus to produce an assembly which is blasted and analyzed. 
It also contains a test dataset which are a subset of 30,000 lines from the input fastq files from InptFiles() function. The SRR IDs for the test data are: SRR5660030 SRR5660033 SRR5660044 SRR5660045


Software Tools Required
=========

* Python3
* Biopython
* Kallisto
* R
* SPAdes
* Bowtie2

==================

pythonwrapper.py
